第一章 统计学习方法概论

这是我阅读李航老师的著作《统计学习方法》一书所做的学习笔记，由于本人水平有限，如有错误请您不吝指教。

入门机器学习领域的教材实在太多了，通过调查我发现主要有两类教材：一类偏重数学的基本定理和公式的推导，另一类偏重实实在在地完成一些机器学习的项目。
显然，李航老师的《统计学习方法》和周志华老师的《机器学习》都是属于前者的。读了两年的研究生，我已深刻体会到掌握理论基础的重要性，没有扎实的数理基础，我将永远是一个幼稚的模仿者。

本章简要叙述统计学习的基本概念。至于“统计学习”和“机器学习”这两个名词，有这样几种看法：①统计学习就是机器学习；②统计学习是机器学习的一个主要分支；③统计学习是机器学习的基础；
其实这些看法没有明确对错之分，从发展历史来看，统计学习是机器学习早期的代表性流派。而且统计学习的各种概念，方法和理论都构成了机器学习理论的基础，从这个角度上说，统计学习和机器学习其实难分彼此。

统计学习

任何学科，都有其特有的研究对象，研究目的和研究方法。而统计学习这个学科的核心就是“数据”，即通过统计学的方法实现对数据的学习。个人认为，统计学习，或说今天的人工智能之所以显得越来越重要，就是因为传统的数据处理和分析的手段已经越来越不能满足先进的需求了。

监督学习

本书只要讨论的是监督学习，这是统计学习极其重要的一个分支，也是应用最广泛的部分。主要有回归任务、分类任务及标注任务。
专业名词解释，见《机器学习》第一章笔记，不再赘述。
监督学习基本假设：①数据存在一定的统计规律，输入变量$X$与输出变量$Y$具有联合概率分布。②训练数据与测试数据是依此联合概率分布独立同分布产生的。

统计学习三要素

统计学习 = 模型 + 策略 + 算法；确定了三要素，也就确定了一个具体的统计学习方法。

模型：

在监督学习中，模型就是要学习的条件概率分布或决策函数，假设空间包含了所有可能的模型。因此统计学习的第一步就是确定问题的假设空间。

策略：

策略指的是在假设空间中选择最优模型的准则，实际中一般用损失函数或风险函数来定量描述这种策略。统计学习的第二步就是选择合适的损失函数或风险函数，通过最小化风险函数，来选择最优模型。

统计学习常用的损失函数有一下几种：
（1）0-1损失函数（0-1 loss function）

（2）平方损失函数（quadratic loss function）

（3）绝对损失函数（absolute loss function）

（4）对数损失含数（logarithmic loss function）或对数似然损失函数（log-likelihood loss function）

损失函数亦称代价函数，损失函数的期望值称为风险函数或期望损失，风险函数很难直接求得，一般用模型关于训练集的平均损失来作为风险函数的近似，称为经验风险。

经验风险最小化与结构风险最小化：
经验风险最小化策略认为使经验风险最小的模型是最优模型。典型的例子有极大似然估计（maximum likelihood estimation）。但是，该策略可能会出现过拟合现象。
结构风险最小化策略是为了防止过拟合提出的，等价于正则化（regularization）。即在经验风险上加上表示模型复杂度的正则化项（regularizer）或罚项（penalty）。

算法：

求解最小化风险函数的具体计算方法，统称为算法。例如牛顿法，梯度下降法，遗传算法等等，都是常见的求解全局最优化的数值计算方法。
另外，由于在学习开始之前，我们就确定了模型，策略和算法这三个要素，故称之为监督式学习。

模型评估与模型选择

训练误差的大小，可以判断问题本身是否容易学习，但一般不重要。测试误差反应了学习方法对未知数据的预测能力（泛化能力），是学习中的重要概念。

正则化与交叉验证

正则化符合奥卡姆剃刀原理。从贝叶斯估计的角度来看，正则化项对应模型的先验概率。可以假设复杂的模型有较小的先验概率，简单的模型有较大的先验概率。
假设样本数据充足，进行模型选择的一种方法是将数据随机地切分为三部分，训练集用于训练模型，验证集用于选择模型，测试集用于对学习方法的评估。
当样本数据不充足时，可采用交叉验证的方法。
交叉验证同《机器学习》第一章。

泛化能力

泛化误差即学的模型的风险函数（期望风险）。
泛化误差上界：通过比较两种学习方法的泛化误差上界来比较他们的优劣，它是样本容量的函数，当样本容量增大时，泛化上界趋于0；它是假设空间容量的函数，假设空间容量越大，模型越难学，泛化误差上界就越大。

生成模型与判别模型

监督学习方法又可以分为生成（geberative）方法和判别（discrimination）方法，所学到的模型分别为生成模型和判别模型。

判别方法是根据数据直接学习决策函数$f(X)$或条件概率分布$P(Y|X)$作为预测模型，也就是对于给定的输入$X$，预测出怎样的$Y$的问题。典型的判别模型包括：$k$近邻法、感知机、决策树、逻辑回归、最大熵模型、支持向量机、提升方法和条件随机场等等。

生成方法是根据数据，先学习联合概率分布$P(X,Y)$，然后在求出条件概率分布$P(Y|X)$作为预测模型，生成模型不仅给出对于输入$X$，应该预测怎样的$Y$，还给出了$X$与$Y$之间的生成关系。典型的生成模型有：朴素贝叶斯法和隐马尔科夫模型。
