What is NumPy?

NumPy是一个用于进行科学计算的Python基础包。它提供了多维数组对象和多种派生对象（例如masked arrays和矩阵），以及处理数组的多种多样的方式（数学运算，逻辑运算，形状调整，排序，选择，I/O，离散傅里叶变换，线性代数，统计学运算等等）。
NumPy的核心package就是ndarray对象，注意numpy.array与Pyhton标准库中的array.array是不同的，后者只能处理一维数组。ndarray的独特之处如下：

  1. NumPy arrays的大小在创建是就固定了，修改ndarray的大小将创建一个新的array并删除旧的。
  2. NumPy arrays中的所有元素的数据类型必须是相同的。
  3. NumPy arrays的各种运算的效率比Pyhton内置的更高。

总而言之，用Python做科学计算是离不开NumPy的，它极大地弥补了Python运算速度“慢”的缺点。官网给出这样的例子：两个向量相乘，使用NumPy可以实现接近C语言的运行速度，并且比原生Python的代码更简洁（c=a*b）。

NumPy快速入门

NumPy的主要对象就是多维数组（homogeneous multidimensional array）。【这里的homogeneous似乎可以翻译成“齐次”，但如何理解呢？】

NumPy中的array类被称作ndarray，ndarray重要的属性如下：

  • ndarray.ndim 数组的轴的数目（维数）。
  • ndarray.shape 以元祖的形式给出数组的维度，对于一个M*N的矩阵，其shape就是(M,N)，显然len(shape) = ndim，shape表明了数组在每一个轴方向上的形状。
  • ndarray.size 数组中元素的数目，显然size等于shape中所有元素的乘积
  • ndarray.dtype 数组中元素的数据类型，常见的有int64，float64等。
  • ndarray.itemsize 数组中每个元素的大小（单位为bytes）

arange，linspace函数

arange函数类似Python自带的range函数，通过给定初始值、终值和步长（默认为1）来创建一维数组。当arange使用float型数据时，由于精度限制，可能事先无法预测生成的数组中元素的数目。
使用linspace函数可以避免上述情况，linspace通过给定初始值，终值和元素个数（默认为50）来创建一个数组。linspace经常于测试函数在多个点上的取值。

其他常用函数

介绍几种常用的函数。

reshape函数

reshape函数用来调整数组的形状，注意，由于NumPy数组在创建时形状就已经确定了，所以使用reshape函数将创建一个新的指定形状的数组，并把原数组中的元素填充进去。

random函数

random.random(shape)函数可以用来生成一个指定形状的随机数组。
random.randint(start，end)可以用来生成指定范围内的随机整数。

all函数和any函数

all函数用来测试数组中所有元素在某一轴向上和“与”运算是否为True（axis默认为None，即所有元素的和运算）。all函数可以用来判断两个数组是否相等。

any函数用来测试数组中所有元素在某一方向上的“或”运算是否为True（axis默认值同上）。

这两个函数的意思就是，在某一轴向上，如果所有元素都为True，all返回True。只要有一个元素为True，any返回True。

索引，切片与迭代

一维数组的索引，切片与迭代与Python中的list是完全类似的。

多维数组在每个轴方向上有一个索引，这些索引可用元祖给出（括号可省略），索引与索引之间用逗号隔开。对于二维数组，有两个轴方向axis=0与axis=1，分别表示列方向与行方向。因此，索引（m,n）就表示了列方向上第m个位置与行方向上第n个位置的元素。换句话说，就是m行n列处的那个元素。（列方向上第m个数，就是第m行）。
